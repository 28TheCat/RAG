1. 函数多返回值
Python 支持一个函数返回多个数据，本质上是返回了一个元组 (Tuple)。

返回方式：支持返回不同类型的数据，变量之间直接用逗号隔开即可。

接收方式：按照返回的顺序，写多个对应的变量来接收，变量之间同样用逗号隔开。

Python
def test_return():
    return "王雅婷", 20, True  # 返回不同类型

name, age, is_student = test_return() # 按顺序接收
2. 函数参数传递
掌握多种传参方式，可以让函数调用更加灵活：

位置参数：调用时根据函数定义的参数位置来传递数据（最基本的传参方式）。

关键字参数：通过“键=值”的形式传递参数。可以让函数调用更清晰，且不限参数顺序。

Python
def user_info(name, age):
    print(f"姓名: {name}, 年龄: {age}")

user_info(age=20, name="王雅婷") # 关键字传参，顺序可乱
缺省参数（默认参数）：在定义函数时为参数提供默认值。调用时若不传该参数，则使用默认值；若传了，则覆盖默认值。

不定长参数：

位置传递 (*args)：接收任意数量的位置参数，合并为一个元组。

关键字传递 (**kwargs)：接收任意数量的关键字参数，合并为一个字典。

3. 函数作为参数传递
函数本身可以作为参数传入另一个函数中使用。

核心逻辑：这是一种逻辑传递。你传入的不是具体的数据，而是处理数据的“动作”或“逻辑”。

Python
def compute(x, y, func):
    return func(x, y)

def add(a, b):
    return a + b

print(compute(10, 20, add)) # 将 add 函数作为参数传入
4. Lambda 匿名函数
定义方式：

def 关键字：用于定义带有名称的函数，支持多行代码，可重复使用。

lambda 关键字：用于定义匿名函数（无名称）。

形式参数：例如 lambda x, y: x + y，其中 x, y 表示接收的两个输入数据。

使用场景：适用于只需临时构建、只用一次的简单场景。

核心限制：

匿名函数内部只能写一行代码。

如果函数体逻辑复杂，需要写多行代码，则不能使用 lambda，必须使用 def 定义。
