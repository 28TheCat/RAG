一、初识对象
1. 什么是对象？

👉 对象 = 数据 + 行为（功能）

现实例子：

🐱 猫
属性：名字、年龄、颜色
行为：叫、吃、跑
2. 数据的组织方式
🟡 生活中
用“对象”描述事物
每个事物都有属性和行为
🟡 程序中

方式1：用变量（不好）

name = "Tom"
age = 3

👉 ❌ 数据分散，不好管理

方式2：用对象（推荐 ⭐）

class Cat:
    pass

cat1 = Cat()
cat1.name = "Tom"
cat1.age = 3

👉 ✔ 数据集中，更清晰

二、类与对象
1. 类（Class）

👉 类 = 模板 / 图纸

class 类名:
    pass
2. 对象（Object）

👉 对象 = 类创建出来的实例

对象 = 类名()
三、成员变量和成员方法
1. 成员变量（属性）
class Cat:
    def __init__(self):
        self.name = "Tom"
        self.age = 3

👉 self.xxx 就是成员变量

2. 成员方法（函数）
class Cat:
    def say(self):
        print("喵喵喵")
四、成员方法语法 ⭐
class 类名:
    def 方法名(self, 参数):
        方法体
示例
class Cat:
    def say(self):
        print("喵喵喵")

    def eat(self, food):
        print(f"正在吃 {food}")
调用方法
cat = Cat()
cat.say()
cat.eat("鱼")
五、注意事项 ⚠️
1. 必须有 self
def say(self):

👉 表示“当前对象”

2. 调用时不用传 self
cat.say()  # 自动传入 self
3. 成员变量必须用 self
self.name = "Tom"
六、类与现实世界

👉 类就是对现实事物的抽象

示例：学生类
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def study(self):
        print(f"{self.name} 正在学习")
七、用类描述现实事务 ⭐
示例：狗
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name} 在汪汪叫")

    def run(self):
        print(f"{self.name} 在跑")
八、基于类创建对象
dog1 = Dog("旺财", 2)
dog1.bark()
dog1.run()
九、构造方法（init）⭐
作用

👉 创建对象时自动执行

语法
def __init__(self, 参数):
示例
class Cat:
    def __init__(self, name, age):
        self.name = name
        self.age = age
创建对象时传参
cat1 = Cat("小白", 2)
十、属性赋值
1. 构造方法中赋值（推荐 ⭐）
class Cat:
    def __init__(self, name):
        self.name = name
2. 对象创建后赋值
cat = Cat("Tom")
cat.age = 3
🐱 完整案例（综合）
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def introduce(self):
        print(f"我叫{self.name}，今年{self.age}岁")

    def study(self, subject):
        print(f"{self.name} 正在学习 {subject}")


# 创建对象
stu1 = Student("小明", 18)

# 调用方法
stu1.introduce()
stu1.study("Python")
