**📦** **1.** **列表 (List)：全能工具箱**

**定义**：my_list = [元素1, 元素2, ...]

**特点**：有序、可变、支持混装、允许重复。

**🛠****️** **常用操作表**

| **功能**       | **语法**                       | **备注**                     |
| -------------- | ------------------------------ | ---------------------------- |
| **查询索引**   | index =  my_list.index(元素)   | 找不到会报错                 |
| **修改元素**   | my_list[索引] = 新值           | 直接覆盖                     |
| **插入元素**   | my_list.insert(索引, 元素)     | 在指定位置插入，后续元素后移 |
| **追加元素**   | my_list.append(元素)           | 加在最后（最常用）           |
| **追加序列**   | my_list.extend(容器)           | 把另一个容器的内容拆开加进去 |
| **删除(索引)** | del my_list[索引] 或 pop(索引) | pop 会返回被删掉的值         |
| **删除(匹配)** | my_list.remove(元素)           | 仅删除**第一个**匹配项       |
| **清空**       | my_list.clear()                | 变为空列表 []                |
| **统计频率**   | my_list.count(元素)            | 统计出现次数                 |
| **统计总数**   | len(my_list)                   | 列表的长度                   |



**🔒** **2.** **元组 (Tuple)：只读列表**

**定义**：my_tuple = (元素1, 元素2, ...)

**特点**：有序、**不可变**、支持重复。

**⚠****️** **注意事项**

- **单元素陷阱**：定义只有一个元素的元组必须加逗号：t1 = ("hello", )。如果不加，它就是一个普通的字符串。
- **“****伪”不可变**：如果元组内部嵌套了一个列表，列表里的内容是可以改的，但元组本身的指向不能变。



**📝** **3.** **字符串 (String)：字符的排队**

**定义**：my_str = "content"

**特点**：有序、**不可变**（修改操作都会返回新字符串）。

**🛠****️** **常用操作**

- **查找**：my_str.index("sub")

- **替换**：my_str.replace(旧, 新) (返回新字符串)

- **分割**：my_str.split("分隔符")     (返回列表)

- **规整 (Strip)**：

- - strip()：去前后空格。
  - strip("12")：去掉前后指定的 "1" 和 "2"。

- **统计**：len(my_str) 或 my_str.count("a")



**✂****️ 4.** **序列切片 (Slicing)**

这是列表、元组、字符串通用的高级操作。

**语法**：序列[起始:结束:步长]

- **起始**：包含。
- **结束**：**不包含**（留头不留尾）。
- **步长**：默认为 1。如果是 -1，表示**反向取值**。

**💡** **常用范例：**

Python

s = "0123456"

 

\# 1. 截取中间

print(s[1:4])   # "123"

 

\# 2. 从头开始到某位

print(s[:3])   # "012"

 

\# 3. 隔一个取一个

print(s[::2])   # "0246"

 

\# 4. 字符串翻转 (面试高频)

print(s[::-1])  # "6543210"

 

\# 5. 复制整个序列

new_list = my_list[:] 



**📁** **5.** **文件操作 (File I/O)**

记住三个步骤：**打开 -> 读/写 -> 关闭**。

**🔓** **打开模式**

- 'r'：只读（默认，文件不存在会报错）。
- 'w'：写入（**覆盖**原内容，不存在则创建）。
- 'a'：追加（在末尾写入，不存在则创建）。

**📖** **读取与写入**

Python

\# 使用 with 语句，会自动帮你 close()，防止内存泄漏

with open("test.txt", "r", encoding="UTF-8") as f:

  \# 读取全部

  content = f.read() 

  \# 读取一行

  line = f.readline()

  \# 读取所有行存入列表

  lines = f.readlines() 

 

with open("test.txt", "a", encoding="UTF-8") as f:

  f.write("\nHello Python!") # 写入

  f.flush() # 刷新缓冲区，真正写入硬盘
