一、文件与编码基础
1. 什么是文件
文件是存储在计算机中的数据集合（文本、图片、视频等）。
本质：一串 0 和 1（二进制）
人类可读内容（如文本）需要通过**编码（encoding）**转换成 0/1 存储。
2. 常见编码
utf-8（最常用，推荐）
gbk（中文 Windows 常见）
ascii
3. 查看文件编码（简单方法）
# 使用第三方库 chardet
import chardet

with open("test.txt", "rb") as f:
    data = f.read()
    print(chardet.detect(data))
二、文件操作基本流程
打开文件（open）
读 / 写 文件
关闭文件（close）
三、open() 打开文件
open(file, mode, encoding)
常用参数：
file：文件路径
mode：模式
encoding：编码（文本文件必须写）
1. 常用三种基础模式
模式	说明
r	只读
w	写入（覆盖）
a	追加
示例
f = open("test.txt", "r", encoding="utf-8")
四、文件读取
1. read() —— 读取全部内容
f = open("test.txt", "r", encoding="utf-8")
content = f.read()
print(content)
f.close()
2. readline() —— 读取一行
f = open("test.txt", "r", encoding="utf-8")

line1 = f.readline()
line2 = f.readline()

print(line1)
print(line2)

f.close()
3. readlines() —— 读取所有行（列表）
f = open("test.txt", "r", encoding="utf-8")

lines = f.readlines()
print(lines)

f.close()
4. for 循环逐行读取（推荐 ⭐）
f = open("test.txt", "r", encoding="utf-8")

for line in f:
    print(line.strip())

f.close()
五、关闭文件
f.close()

⚠️ 不关闭可能导致：

数据未写入
占用资源
六、with open（推荐 ⭐⭐⭐）

自动关闭文件，更安全！

with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)
七、文件写入
1. 写入（w 模式）
with open("test.txt", "w", encoding="utf-8") as f:
    f.write("Hello World\n")
    f.write("Python 文件写入")

⚠️ 注意：

w 会覆盖原内容
2. 追加（a 模式）
with open("test.txt", "a", encoding="utf-8") as f:
    f.write("\n这是追加的内容")
3. 写入多行
lines = ["第一行\n", "第二行\n", "第三行\n"]

with open("test.txt", "w", encoding="utf-8") as f:
    f.writelines(lines)
八、文件操作注意事项 ⚠️
1. 编码必须一致
# 写入 utf-8，读取也必须 utf-8
2. 文件路径问题
# 相对路径
open("test.txt")

# 绝对路径
open("C:/Users/xxx/test.txt")
3. 模式选择错误
r 文件不存在 → 报错
w 会清空文件
a 不会覆盖
4. 换行问题
f.write("hello\n")
九、完整示例（读 + 写）
# 写入文件
with open("demo.txt", "w", encoding="utf-8") as f:
    f.write("第一行\n")
    f.write("第二行\n")

# 读取文件
with open("demo.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

文件本质：0 和 1
